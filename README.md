AppWorks Blockchain Course
===

記錄上課練習的合約，主要搭配

- yarn workspace
- Solidity 合約：hardhat
- 前端：Vite + React + TypeScript
- Code formatter：Prettier

## 建置

```
yarn
```

PS. 想從零開始建置 hardhat ，可以參考 [Week3](../week3/README.md)

## 執行

根目錄執行
```
yarn workspace @app-block/week3 compile
```

也可以到各目錄內執行 script
